## Usage

A typical conversion flow looks like this:

1. Open Acme CAD Converter.
2. Add one file or a full batch of CAD drawings.
3. Choose the source and destination formats, such as DWG to PDF or DXF to SVG.
4. Keep layer and metadata options enabled when those details matter to the output.
5. Start the conversion and review the generated files in the output folder.

Example batch routine:

- Select a directory of drawings
- Set the output format once
- Apply the same export settings to every file
- Run the batch and collect the converted results in a target folder
